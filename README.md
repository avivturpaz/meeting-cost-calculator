# Meeting Cost Calculator

> **Retired Venture experiment — historical reference only.**

A small Flask prototype that estimates the running cost of a meeting from the number of participants, hourly compensation, and elapsed time.

## What is implemented

- Live meeting timer and cost calculation
- Participant and hourly-rate inputs
- SQLite session storage
- Basic web interface
- Paddle checkout interface

## Status

Development stopped on July 10, 2026. The application is not maintained or production-ready. Calculations are simple estimates and do not account for benefits, overhead, taxes, preparation, or follow-up time.

Created as part of the archived Venture product-experiment portfolio.
